# Security Policy

## Supported Versions

This project is pre-1.0 and evolves rapidly. Security fixes are applied to the latest `main` branch.

## Reporting a Vulnerability

Please do not open a public issue for suspected security vulnerabilities.

Instead, report privately to the maintainers with:

- impact summary
- reproduction steps
- affected versions or commit
- suggested fix (optional)

You will receive acknowledgement and triage guidance as quickly as possible.

## Scope Notes

GSS packages define protocol contracts and orchestration. Operational secrets and persistence are expected to be managed by webshop implementations using these packages.
